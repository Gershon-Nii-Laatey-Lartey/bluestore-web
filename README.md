Bluestore is a structural, world-class marketplace discovery engine engineered for high-density industrial-minimalist exploratory commerce. Featuring an authoritative reputation rail, situational property discovery, and merchant-centric inventory management.

## 🌟 Modernization Milestones

*   **🏢 Industrial-Minimalist Aesthetic**: Orchestrated a world-class visual identity featuring **Outfit** premium typography and a zero-noise layout.
*   **🍱 Sidebar Collapse Rail**: High-density scaling ($220px$ to $72px$) providing absolute situational awareness and maximum discovery real estate.
*   **🏷️ Situational Badge Stack**: Ground-truthed discovery labels for *Boosted* and *Urgent* properties, synchronized with live database metadata.
*   **🛠️ Merchant Lifecycle Engine**: Integrated high-fidelity *Update Property* and *Decommissioning* triggers, allowing merchants to definitive-manage their inventory footprints.
*   **💬 Communication Hub**: Aerodynamic chat interface featuring transactional trust guardrails and verified merchant identity.
*   **📍 Verified Discovery**: Comprehensive location-aware search and filtering calibrated for the storefront exploratory journey.

## 🚀 Tech Stack & Infrastructure

*   **Logic**: React 19 + TypeScript 5
*   **Orchestration**: Vite 6
*   **Data Lake**: Supabase (PostgreSQL + RLS + Storage)
*   **Visual Identity**: Vanilla CSS + Lucide Icons + Framer Motion
*   **Deployment**: Vercel (Edge-ready)

## 📦 Installation & Exploratory Setup

1.  **Clone the Discovery Rail**:
    ```bash
    git clone https://github.com/Gershon-Nii-Laatey-Lartey/bluestore-web.git
    cd bluestore-web
    ```

2.  **Synchronize Dependencies**:
    ```bash
    npm install
    ```

3.  **Calibrate Environment**:
    Create a `.env` file with your authoritative Supabase credentials:
    ```env
    VITE_SUPABASE_URL=your_project_url
    VITE_SUPABASE_ANON_KEY=your_anon_key
    ```

4.  **Initiate Discovery**:
    ```bash
    npm run dev
    ```
